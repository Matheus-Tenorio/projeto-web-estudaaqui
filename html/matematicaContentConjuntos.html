Conteúdos de Matemática

O que são conjuntos?

Um conjunto é estabelecido quando agrupamos elementos com as mesmas características. Esses agrupamentos possuem notação própria, utilizando-se letras maiúsculas para dar nome a eles, e representação específica, em geral por meio de círculos, formando-se o que se conhece como diagrama de Venn, ou listando-se os elementos dos conjuntos

Tipos de conjuntos

  • Conjunto vazio: Não possui nenhum elemento. Representado por { } ou ∅.
  • Conjunto finito: Possui um número definido de elementos.
  • Conjunto infinito: Possui infinitos elementos, como o conjunto dos números naturais.

Operações com conjuntos

As principais operações entre conjuntos são:

  • União: A U B - todos os elementos de A e B.
  • Interseção: A ∩ B - apenas os elementos comuns entre A e B.
  • Diferença: A - B - elementos que estão em A, mas não em B.
